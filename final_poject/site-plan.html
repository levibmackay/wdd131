Overview

Purpose

The purpose of this website is to help beginners build custom workout routines and stay consistent with their fitness goals. It provides tools to plan workouts, switch between gym and home training, and stay motivated with quotes and mindset support.

Audience

College students and young adults who want to get in shape but don’t know where to start, or who want an easy way to keep track of workouts and motivation without paying for expensive apps.

Dynamic elements

- JavaScript will be used to display and filter workout routines based on gym or home selection.
- Workouts will be stored in arrays of objects and dynamically rendered with DOM manipulation.
- A form will allow users to save a custom plan or workout goal to localStorage.
- The motivation page will include a random quote generator and a goal setter with localStorage support.

Content

Home page

The home page will feature a hero banner with the logo and tagline, a short welcome blurb, and a toggle or dropdown menu to choose between "Home Workouts" or "Gym Workouts." Based on the selection, workouts will display below in card format with name, sets/reps, and a short description. A button will allow the user to "Add to Plan," saving the selection to localStorage. Responsive layout and modern CSS styling will ensure readability and accessibility on all devices.

Motivation Page

The second page will focus on motivation and mindset. A JavaScript-powered quote generator will show a new quote each time the page is loaded or when the user clicks a button. Below, a short list of mental tips (e.g., “Discipline over motivation”) will be shown. The user can enter a fitness goal into a form that saves to localStorage and is displayed beneath the input. This page is built to inspire and help users stay committed.
